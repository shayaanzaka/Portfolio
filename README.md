## Overview

A polished, production-ready portfolio designed to highlight professional achievements. It beautifully showcases:

* **Education**: Academic background and degrees.
* **Experience**: Professional journey and career milestones.
* **Skills**: Technical proficiencies and tools.
* **Projects**: Featured work with links to repositories and live demos.
* **Resume**: Downloadable PDF version of the resume.
* **Contact information**: Easy ways to connect professionally.

---

```bash

```

**Production Build:**

```bash
npm run build
```

**Production Server:**

```bash

```

## Customization

The portfolio is designed to be easily customizable. Update the following locations with your personal data:

* **Personal Information**: Update within the Hero and Footer components.
* **Projects**: Update the data array inside `components/Projects/Projects.tsx`.
* **Skills**: Update the items within `components/Skills/Skills.tsx`.
* **Experience**: Update the history in `components/Experience/Experience.tsx`.
* **Resume**: Replace `public/resume.pdf` with your own PDF file.
